## Опис завдання
Цей репозиторій містить виконання Лабораторної роботи №3, яка присвячена дослідженню та візуалізації багатовимірних даних. 

В якості основи обрано **Automobile Dataset** з репозиторію машинного навчання UCI. 
* Здійснено попереднє очищення даних (Data Cleaning) засобами `pandas`: заповнення числових пропусків середнім значенням, категоріальних — модою, та видалення критичних пропусків (відсутність ціни).
* Побудовано комплекс візуалізацій (5 завдань) за допомогою `matplotlib` та `seaborn`, що відображають глибокі залежності між атрибутами (включно зі звичайними та багатовимірними Scatterplot, гістограмою розподілу, Boxplot, Countplot та повноцінною Pairplot-матрицею).

## Структура директорії
* `lab3.ipynb` — головний файл із рішенням (код очищення даних та побудова всіх графіків).
* `data/` — папка для зберігання сирого датасету.
* `requirements.txt` — список необхідних залежностей (`pandas`, `matplotlib`, `seaborn` тощо).
* `.gitignore` — файл ігнорування Git.

## Інструкція із запуску
1. Переконайтеся, що встановлено Python 3.10+ (використовувалася версія 3.13).
2. Клонуйте репозиторій та перейдіть у директорію `lab3`.
3. Створіть віртуальне середовище: `python -m venv venv`
4. Активуйте його (для Windows): `venv\Scripts\activate`
5. Встановіть необхідні бібліотеки: `pip install -r requirements.txt`
6. **Увага:** Завантажте датасет *Automobile Dataset* з репозиторію UCI та покладіть головний файл `imports-85.data` у підпапку `data/`.
7. Відкрийте Jupyter Notebook для перегляду та запуску коду: `jupyter notebook`
