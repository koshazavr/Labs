**Виконав:** Андрущенко Костянтин

## Опис завдання
Цей репозиторій містить виконання Лабораторної роботи №5, яка присвячена обміну результатами дослідження та створенню інтерактивних веб-додатків для аналізу даних. 

В якості основи обрано дані NOAA (вегетаційні індекси VCI, TCI, VHI по областях України), отримані в рамках виконання Лабораторної роботи №2. 
* Здійснено автоматизоване зчитування та очищення 27 сирих CSV-файлів (видалення HTML-тегів, обробка висячих ком, фільтрація некоректних значень).
* Створено інтерактивний веб-дашборд за допомогою `Streamlit` із можливістю гнучкої фільтрації (за показником, областю, інтервалом років та тижнів) і сортування даних.
* Побудовано інтерактивні візуалізації за допомогою `plotly.express` (графік динаміки показника та стовпчикова діаграма для порівняння областей).

## Структура директорії
* `lab5.py` — головний файл із рішенням (логіка обробки даних та побудова інтерфейсу).
* `data/` — папка для зберігання сирих CSV-файлів.
* `requirements.txt` — список необхідних залежностей (`pandas`, `streamlit`, `plotly`).
* `.gitignore` — файл ігнорування Git (налаштовано на ігнорування папки `data/`).

## Інструкція із запуску
1. Переконайтеся, що встановлено Python 3.10+ (використовувалася версія 3.13).
2. Клонуйте репозиторій та перейдіть у директорію `lab5`.
3. Створіть віртуальне середовище: `python -m venv .venv`
4. Активуйте його (для Windows): `.venv\Scripts\activate`
5. Встановіть необхідні бібліотеки: `pip install -r requirements.txt`
6. **Увага:** Перенесіть 27 завантажених CSV-файлів NOAA з Лабораторної роботи №2 та покладіть їх у підпапку `data/`.
7. Запустіть веб-додаток: `streamlit run lab5.py`
8. Додаток автоматично відкриється у браузері (зазвичай за адресою `http://localhost:8501`).
