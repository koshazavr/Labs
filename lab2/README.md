## Опис завдання
Цей репозиторій містить виконання Лабораторної роботи №2, яка розділена на два незалежні Jupyter Notebook файли:

* **Частина 1 (`lab2_part1.ipynb`):** Автоматизоване завантаження даних VHI з серверів NOAA за допомогою `urllib`. Очищення даних засобами `pandas`, зміна індексів областей та базовий аналіз часових рядів.
* **Частина 2 (`lab2_part2.ipynb`):** Аналіз датасету "Individual Household Electric Power Consumption". Виконано складні вибірки даних виключно засобами `pandas`. Усі процедури профілюються (`timeit`). Реалізовано нормалізацію та стандартизацію (`scikit-learn`), розрахунок кореляцій та One Hot Encoding.

## Структура директорії
* `lab2_part1.ipynb` — рішення першої частини.
* `lab2_part2.ipynb` — рішення другої частини.
* `requirements.txt` — список необхідних залежностей.
* `.gitignore` — файл ігнорування Git.

## Інструкція із запуску
1. Переконайтеся, що встановлено Python 3.10+ (використовувалась версія 3.13).
2. Клонуйте репозиторій та перейдіть у директорію `lab2`.
3. Створіть віртуальне середовище: `python -m venv venv`
4. Активуйте його (для Windows): `venv\Scripts\activate`
5. Встановіть необхідні бібліотеки: `pip install -r requirements.txt`
6. **Увага (для Частини 2):** Завантажте датасет *Individual Household Electric Power Consumption* та розпакуйте `household_power_consumption.txt` у корінь папки `lab2` (файл ігнорується Git через великий розмір).
7. Відкрийте Jupyter Notebook для запуску коду: `jupyter notebook`
