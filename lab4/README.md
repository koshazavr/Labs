# Лабораторна робота №4: Візуалізація даних 2 (Інтерактивна обробка сигналів)

**Виконав:** Андрущенко Костянтин
**Група:** ФБ-44

## Опис програми
Програма моделює гармонічний сигнал (синусоїду) із накладенням нормально розподіленого (Гаусівського) шуму. До зашумленого сигналу застосовується цифровий фільтр низьких частот (Low-pass Butterworth filter) для відновлення початкової "чистої" гармоніки.

## Інструкція користувача
Після запуску скрипта відкриється інтерактивне вікно з графіком та панеллю керування знизу.

**Як користуватися:**
1. **Слайдери гармоніки (Амплітуда, Частота, Фаза):** Змінюють параметри базової "чистої" синусоїди (синій пунктир). Зверніть увагу, що при зміні цих параметрів структура накладеного шуму залишається незмінною.
2. **Слайдери шуму (Середнє шуму, Дисперсія):** Керують характеристиками випадкового шуму. Дисперсія відповідає за "розмах" відхилень.
3. **Слайдер фільтрації (Частота зрізу):** Визначає, наскільки агресивно фільтр відсікає високі частоти шуму. Зменшуйте це значення, щоб згладити фіолетовий графік (відфільтрований сигнал), але не робіть його меншим за частоту самої гармоніки, інакше сигнал спотвориться.
4. **Чекбокс "Показати шум":** Дозволяє приховати помаранчевий "сирий" графік, щоб чітко побачити, наскільки добре відфільтрована лінія (фіолетова) накладається на чистий ідеал (синя пунктирна).
5. **Кнопка Reset:** Повертає всі повзунки та налаштування у початковий, ідеально збалансований стан.
